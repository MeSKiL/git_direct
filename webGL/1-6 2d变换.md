# 2d变换
## 平移
- 二维空间可以用平移向量(tx,ty)表达
- 齐次坐标，需要增加一个维度,最后的矩阵是 
```
[1,0,Tx
0,1,Ty
0,0,1]
```
- 平移的操作可以看作一个矩阵，产生这个矩阵的函数，功能上类似一个高阶函数

## 世界 -> 剪裁空间
```
[2/w,0,-1,
0,2/h,-1,
0,0,1]
```

## 拉伸
```
[Sx,0,0,
0,Sy,0
0,0,1]
```

## 旋转
```
[Math.cos(a), -Math.sin(a), 0,
Math.sin(a), Math.cos(a), 0,
0, 0, 1]
```
